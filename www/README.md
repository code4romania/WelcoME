## Prerequisite
###  Node js
 https://nodejs.org
### Git
 https://git-scm.com
### yarn (optional) 
 https://yarnpkg.com
### React storybook
 https://storybooks.js.org/

## Installation
* `git clone https://github.com/code4romania/WelcoME.git`
* `cd WelcoME`
* `cd www`
* `npm i _or_ yarn`
* `cd ../functions`
* `npm i _or_ yarn`

## Run
* `cd www`
* `npm start _or_ yarn start`

## Storybook
* `cd www`
* `npm run storybook _or_ yarn run storybook`
