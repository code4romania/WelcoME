# WelcoME

install web:
`cd www && npm install`

install functions
`cd functions && npm install`

run web:
`cd www && npm start`
and go to:
http://localhost:3000
react guide:
https://blog.tighten.co/react-101-building-a-gif-search-engine

test web:
`cd www && npm run test`

test functions
`cd functions && npm run test`


deploy functions: 
`cd functions && npm run deploy`


build web:
`cd www && npm run build`

deploy web:
`cd www && npm run deploy`
